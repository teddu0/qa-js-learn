Если ты читаешь это, значит ты уже сколнировал проект к себе на устройство.

Для дальнейшей работы тебе необходимо сделать следующее:

1. Проверить установлена ли NODE JS на устройсте, для этого просто введи в терминале команду node -v
    - если NODE JS установлена то ты увидишь в ответ что-то вроде v20.10.0 (версия может отличаться),
    - если вернулась ошибка, то необходимо скачать NODE JS здесь: https://nodejs.org/en/download/package-manager
    - после установки желательно перезапустить устройство, после чего снова в терминале введи команду node -v и увидишь версию своей NODE JS

2. Проверить установлен ли пакетный менеджер NPM введя команду в реминале npm -v
    - если пакетный менеджер установлен, в ответ увидишь версию в виде цифр примерно так 10.8.1
    - если вернулась ошибка, необходимо установить пакетный менеджер через команду в терминале npm install -g npm
    - после установки также перезапустить компьютер и проверить версию NPM через команду npm -v


Запуск своего кода.

После выполнения вышеописанного ты можешь написать свою первую программу, но для этого тебе необходимо отдельную ветку.

1. В левом нижнем углу нажми на main
2. В верхней части в выпадающем списке выбери + Create new branch (создать ветку)
3. Введи название ветки латиницей, например (first lesson или свое имя)
4. Нажми ENTER

В результате ты окажешься в своей ветке, в которой можешь писать свою первую программу "Hello World!!!"

Пишем и запускаем программу.

1. Разверни папку со своим именем и открой файл hell.world.js
2. Напиши программу (это уже домашка, подсказывать не буду)
3. Запусти свою программу выполнив следующие шаги:
    - открой терминал и перейди к своей папке через терминал (пример команды: CD konstantin) и нажать ENTER
    - попробуй запустить файл со своей программой через команду: node hello.world.js

Если ты все сделал правильно, в консоль выведется сообщение Hello World!!!

2. В поле с плейсхолдером message введи любой комментарий, как правило пишут описание проделанной работы, например (написал первую программу Hello world!!!)
3. Открой выпадающий список под полем и нажми Commit & Push
